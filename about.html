ℹ️ Acerca de Simple Notepad

Última actualización: 6 de octubre de 2025

Misión

Simple Notepad nace con un objetivo claro: ofrecer un editor de texto plano rápido, privado y accesible desde cualquier dispositivo sin obligar al usuario a registrarse ni entregar datos personales.

¿Por qué existe?

  • Velocidad: Cargar y escribir inmediatamente.
  • Minimalismo: Solo lo esencial para tomar notas y manejar archivos .txt.
  • Privacidad local: El contenido se procesa en tu navegador (localStorage / APIs nativas).
  • Compatibilidad: Funciona en Windows, macOS, Linux, Android e iOS.

Principios del proyecto

🔒 Privacidad Primero

No almacenamos tu texto en servidores. Tú controlas tus archivos.

⚙️ Simplicidad Técnica

HTML + CSS + JavaScript modular sin dependencias pesadas.

🧩 Extensible

Estructura preparada para añadir exportaciones, temas o colaboración futura.

Tecnología

  • Editor: <textarea> optimizado + gestión de estado.
  • Módulos: FileManager, EmojiManager, UI, AdManager, etc.
  • Emojis: Catálogo amplio Unicode cargado dinámicamente.
  • Persistencia: localStorage para recuperación de sesión.

Hoja de ruta (Roadmap)

  • Exportación Markdown / PDF
  • Modo sin distracciones
  • Temas (claro / oscuro / alto contraste)
  • Soporte multi-documento (tabs)
  • Atajos personalizables

El roadmap es orientativo y puede cambiar según sugerencias de usuarios.

Transparencia

El proyecto incluye publicidad para sostener costos, pero se controla el impacto en la experiencia y se respeta el consentimiento de cookies. No se comercializan datos personales.
