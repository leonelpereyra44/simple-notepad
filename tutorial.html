📚 Cómo usar Simple Notepad

Guía rápida para aprovechar nuestro editor de texto online gratuito.

🚀 Empezar a usar

¡Es súper fácil! Solo abre Simple Notepad y comienza a escribir. Tu texto se guarda automáticamente.

📁 Funciones principales

📝 Nuevo

Limpia el editor para empezar un texto desde cero.

📂 Abrir

Ctrl + O - Carga archivos .txt desde tu computadora.

💾 Guardar

Ctrl + S - Descarga tu texto como archivo .txt.

✏️ Renombrar

Haz clic en el lápiz para cambiar el nombre del archivo.

⌨️ Atajos útiles

Atajo Función
Ctrl + NNuevo archivo
Ctrl + OAbrir archivo
Ctrl + SGuardar archivo
TabInsertar espacios
Ctrl + ASeleccionar todo

📱 En móviles

En el móvil, usa el menú de 3 puntos (⋮) para acceder a todas las funciones. La interfaz se adapta automáticamente a pantallas pequeñas.

💡 Consejos rápidos

  • Guardado automático: Tu texto se mantiene aunque cierres el navegador
  • Funciona offline: Una vez cargada la página, no necesitas internet para escribir
  • Buscar texto: Usa Ctrl + F para buscar dentro de tu documento
  • Compatible: Funciona en Chrome, Firefox, Safari, Edge y móviles
